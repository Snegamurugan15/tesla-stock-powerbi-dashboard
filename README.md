# Tesla Stock Power BI Dashboard

This repository contains a Power BI dashboard for Tesla stock-price analysis. The report focuses on trading volume, open/high/low/close price behavior, monthly trends, conditional formatting, and interactive finance visuals.

## Analytics Included

- Power BI dashboard with Tesla stock visuals and conditional formatting.
- PDF export of the Power BI dashboard for quick viewing.
- Monthly trading-volume comparison.
- Opening, low, high, and closing price summaries.
- Volume and price relationship visuals.
- Year/month filtering for dashboard exploration.

## Files

- `powerbi/tesla-stock-dashboard.pbix` - original Power BI Desktop report.
- `reports/tesla-stock-powerbi-report.pdf` - exported report snapshot.

## Open

Open `powerbi/tesla-stock-dashboard.pbix` in Microsoft Power BI Desktop.

For a quick static preview, open `reports/tesla-stock-powerbi-report.pdf`.
